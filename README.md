# 🚀 Lógica de Programação com VisuAlg

Repositório dedicado ao armazenamento de algoritmos e exercícios práticos desenvolvidos durante os estudos de lógica de programação. Os arquivos utilizam a extensão `.alg`, compatível com o **VisuAlg**.

## 📂 Estrutura do Repositório

O projeto está organizado por categorias para facilitar o aprendizado e a consulta:

* **01-Estruturas-Sequenciais**: Exercícios básicos envolvendo entrada, processamento e saída de dados.
* **02-Estrutura-Condicional**: Algoritmos que utilizam desvios lógicos (SE-ENTÃO-SENÃO).
* **03-Estruturas-Repetitivas**: Aplicação de laços de repetição (ENQUANTO, PARA).
* **04-Vetores-e-Matrizes**: Manipulação de estruturas de dados indexadas.

## 🛠️ Tecnologias Utilizadas

* **VisuAlg 3.0**: Interpretador de algoritmos.
* **Portugol**: Linguagem estruturada em português.

## 📝 Como utilizar

1. Faça o download e instale o [VisuAlg](https://visualg3.com.br/).
2. Clone este repositório ou baixe o arquivo `.alg` desejado.
3. Abra o arquivo no VisuAlg e pressione `F9` para executar.
